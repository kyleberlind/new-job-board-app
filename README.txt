How to Run Locally:

- front end -
From new-job-board-app run 'npm start'. This will start the react app on http://localhost:3000/

- backend -
From new-job-board-api run
1. pipenv shell
2. export FLASK_APP=api
3. export FLASK_DEBUG=1
4. 'flask run'

This will start the flask backend in the virtual environment on http://localhost:5000/
